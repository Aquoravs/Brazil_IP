# Advisor Meeting Trackers

One file per weekly meeting with advisors. Each file classifies comments raised in that meeting, routes them to agents, and tracks status.

## Conventions

- **Filename:** `YYYY-MM-DD_tracker.md` (date of the meeting)
- **Source:** link to the meeting notes in `master_supporting_docs/meetings/`
- **Scope:** items raised in *that* meeting only
- **Carry-over:** manual. Unresolved items are not auto-copied to the next meeting's tracker. If an item resurfaces, reference it explicitly (e.g., `(see 2026-04-17, C4)`).
